# Multi-Threaded OpenCV Camera Display

## Requirements

-   OpenCV C++ library
-   Intels TBB C++ library

## Instructions

```shell
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build .
$ ./opencv_multiple
```
